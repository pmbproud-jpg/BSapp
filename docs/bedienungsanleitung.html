1 Anmeldung und Navigation

Anmeldung

  1. Öffnen Sie die BSapp auf dem Handy oder im Browser.
  2. Geben Sie Ihre E-Mail-Adresse und Ihr Passwort ein.
  3. Klicken Sie auf „Anmelden".
  4. Nach der Anmeldung sehen Sie das Dashboard (Übersicht).

Navigation — untere Leiste

Am unteren Bildschirmrand befinden sich 4 Tabs:

SymbolNameBeschreibung
🏠DashboardÜbersicht mit Statistiken und Schnellzugriff
💼ProjekteListe aller Baustellen/Projekte
👥BenutzerMitarbeiterliste (nur Admin/Geschäftsführung/PM)
⚙️EinstellungenProfil, Sprache, Design, Berechtigungen

Kopfzeile

Am oberen Bildschirmrand sehen Sie:

  • Logo und Firmenname
  • Name und Rolle des angemeldeten Benutzers
  • 🔔 Glocke — Benachrichtigungen (roter Punkt = neue)
  • 🚪 Abmelden — Aus der App abmelden

⏳ Offene AufgabenAufgaben mit Status „zu erledigen" oder „in Bearbeitung"

Fortschrittsanzeige

Ein Kreisdiagramm zeigt den Prozentsatz der erledigten Aufgaben im Verhältnis zu allen.

Schnellzugriff-Buttons

Aufgaben nach Projekt

Am unteren Rand des Dashboards befindet sich eine Liste der Projekte mit Fortschrittsbalken. Man kann nach Name oder Baunummer suchen und sortieren.

3 Projekte (Baustellen)

Projektliste

Der Tab Projekte zeigt alle Baustellen. Sie können:

  • Suchen nach Name oder Baunummer
  • Filtern nach Status (aktiv, Planung, abgeschlossen, pausiert)
  • Auf ein Projekt klicken, um Details zu sehen

Neues Projekt erstellen

  1. Klicken Sie auf „+ Projekt hinzufügen" (im Dashboard oder in der Projektliste).
  2. Füllen Sie die Daten aus: Name, Baunummer, Standort, Beschreibung.
  3. Wählen Sie den Projektleiter (PM) und den Bauleiter (BL).
  4. Klicken Sie auf „Speichern".

Projektdetails

Nach dem Klick auf ein Projekt sehen Sie:

BereichBeschreibung
InformationenName, Nummer, Standort, Status, PM, BL
TeamListe der zugewiesenen Mitarbeiter mit Rollen
AufgabenAufgabenliste mit Filtern (alle/meine/zu erledigen)
BauplänePlanbilder mit Pins (Markierungen)
AnhängeDateien, Fotos, Dokumente in Ordnern

Baupläne

  1. Gehen Sie in das Projekt → Tab „Pläne".
  2. Fügen Sie einen Plan hinzu (Bild/Datei).
  3. Klicken Sie auf „Pin hinzufügen" und klicken Sie auf den Plan, um eine Markierung zu setzen.
  4. Geben Sie eine Beschreibung ein und speichern Sie.
  5. Pins werden als farbige Punkte auf dem Plan angezeigt.

4 Aufgaben

Aufgaben sind immer mit einem bestimmten Projekt verknüpft. Der Zugriff erfolgt über die Projektdetails.

Aufgabe erstellen

  1. Gehen Sie in das Projekt.
  2. Klicken Sie auf „+ Aufgabe hinzufügen".
  3. Füllen Sie aus: Titel, Beschreibung, Priorität, Frist.
  4. Weisen Sie einen Mitarbeiter (oder mehrere) zu.
  5. Klicken Sie auf „Erstellen".
  6. Optional: Fügen Sie Anhänge hinzu (Fotos, Dateien).

Aufgabenstatus

StatusBedeutungFarbe
Zu erledigenAufgabe wartet auf Bearbeitunggelb
In BearbeitungJemand arbeitet an der Aufgabeblau
ErledigtAufgabe abgeschlossengrün
BlockiertAufgabe kann nicht fortgesetzt werdenrot

Aufgabendetails

Nach dem Klick auf eine Aufgabe können Sie:

  • Status ändern (auf den aktuellen Status klicken)
  • Titel, Beschreibung, Priorität, Frist bearbeiten
  • Einen Kommentar hinzufügen
  • Anhänge hinzufügen (Kamera, Galerie oder Dateien)
  • Den zugewiesenen Mitarbeiter ändern

5 Wochenplan

Der Wochenplan dient zur Planung der Mitarbeiterzuweisungen zu Baustellen für jeden Wochentag.

Zugriff

Dashboard → Button „Wochenplan" (blau).

Wochennavigation

Die Pfeile ◀ ▶ wechseln zwischen Wochen. Der Datumsbereich wird angezeigt (z.B. 17.02 – 23.02).

Tabs

TabBeschreibung
WochenplanTagesraster + Zuweisungen + Fahrzeuge
BedarfBL-Bestellungen — Mitarbeiterbedarf
KalenderKalenderansicht

Tagesraster

Jeder Wochentag (Mo–So) zeigt:

  • Blaue Zahl — Anzahl der Zuweisungen für diesen Tag
  • Rote Zahl mit ✗ — Anzahl der abwesenden Mitarbeiter

Klicken Sie auf einen Tag, um die Details zu sehen.

Tagesansicht

Nach dem Klick auf einen Tag sehen Sie:

🚗 Fahrzeuge

Liste der Fahrzeuge mit Sitzplatzbelegung. Sie können Fahrzeuge hinzufügen, bearbeiten und löschen.

🔴 Abwesenheiten

Rote Karte mit der Liste der an diesem Tag abwesenden Mitarbeiter (genehmigte Urlaube + Krankmeldungen). Jeder Eintrag zeigt den Abwesenheitstyp als farbiges Badge.

📋 Zuweisungen

Liste der Zuweisungen gruppiert nach Projekt. Jede Zuweisung zeigt: Mitarbeiter, Fahrzeug, Arbeitszeiten.

Zuweisung hinzufügen

  1. Gehen Sie in die Tagesansicht und klicken Sie auf „+ Zuweisung hinzufügen".
  2. Wählen Sie das Projekt (Baustelle).
  3. Wählen Sie Fahrzeug(e).
  4. Stellen Sie die Arbeitszeiten (von–bis) und die Abfahrtszeit ein.
  5. Wählen Sie Mitarbeiter.
  6. Klicken Sie auf „Speichern".
Das System prüft automatisch Zeitkonflikte — wenn ein Mitarbeiter bereits in diesen Stunden zugewiesen ist, wird eine Warnung angezeigt.

BL-Bedarf

Der Bauleiter kann den Mitarbeiterbedarf melden:

  1. Wechseln Sie zum Tab „Bedarf".
  2. Klicken Sie auf „Neuer Bedarf".
  3. Wählen Sie Projekt, Mitarbeiter, Fahrzeuge und fügen Sie Notizen hinzu.
  4. Klicken Sie auf „Senden".

6 Urlaub und Abwesenheiten

Das Urlaubssystem ermöglicht es Mitarbeitern, Anträge zu stellen, und Vorgesetzten, diese zu genehmigen.

Urlaubsantrag stellen

  1. Gehen Sie zu Benutzer → Mitarbeiterprofil.
  2. Scrollen Sie zum Bereich „Abwesenheiten / Urlaub".
  3. Klicken Sie auf „+ Eintragen".
  4. Wählen Sie den Typ der Abwesenheit.
  5. Wählen Sie die Daten (Von – Bis) durch Klick auf das Datumsfeld — ein Kalender öffnet sich.
  6. Optional: Fügen Sie eine Notiz hinzu.
  7. Klicken Sie auf „Speichern".

Abwesenheitstypen

TypFarbeGenehmigungSichtbarkeit im Plan
UrlaubrotGenehmigung erforderlichNach Genehmigung
KrankmeldunggelbAutomatischSofort
SonderurlaublilaGenehmigung erforderlichNach Genehmigung
SchulungblauGenehmigung erforderlichNach Genehmigung
UnentschuldigtgrauGenehmigung erforderlichNach Genehmigung
Krankmeldungen werden automatisch genehmigt und sind sofort im Wochenplan sichtbar. Alle anderen Typen erfordern eine Genehmigung.

Urlaubskonto

Im Mitarbeiterprofil werden 3 Zähler angezeigt:

  • Resturlaub — wie viele Urlaubstage noch übrig sind
  • Genommen — wie viele Tage bereits genommen wurden
  • Gesamt — Jahresanspruch (Standard: 26 Tage)

Monatskalender

Im Mitarbeiterprofil wird ein Kalender mit farbigen Punkten an Tagen mit Abwesenheiten angezeigt. Die Farblegende befindet sich unter dem Kalender.

Anträge genehmigen

Wer kann genehmigen: Admin, Geschäftsführung, Logistik

  1. Klicken Sie im Dashboard auf den gelben Button „Urlaubsanträge".
  2. Sie sehen die Liste der Anträge mit dem Filter „Ausstehend".
  3. Klicken Sie bei jedem Antrag auf ✓ Genehmigen oder ✗ Ablehnen.
  4. Nach der Genehmigung erscheint der Urlaub im Wochenplan als Abwesenheit.
Die rote Zahl am Button „Urlaubsanträge" im Dashboard zeigt, wie viele Anträge auf Genehmigung warten.

7 Lager

Das Lager dient zur Verwaltung von Materialien und Werkzeugen.

Zugriff

Dashboard → Button „Lager" (rot).

Funktionen

  • Artikelliste — Suche, Sortierung
  • Hinzufügen — klicken Sie auf „+ Artikel hinzufügen"
  • Bearbeiten — klicken Sie auf einen Artikel
  • Löschen — Papierkorb-Symbol
  • Excel-Import — Massenimport aus Datei

Artikeldaten

Jeder Artikel enthält: Beschreibung, Menge, Einheit, Kategorie, Lagerort, Bestellnummer.

9 Benachrichtigungen

Klicken Sie auf die 🔔 Glocke in der Kopfzeile, um die Benachrichtigungen zu sehen.

Benachrichtigungstypen

  • Neue Zuweisung — Sie wurden einer Baustelle im Wochenplan zugewiesen
  • Neue Aufgabe — Ihnen wurde eine Aufgabe zugewiesen
  • Neues Projekt — Sie wurden einem Projektteam hinzugefügt

Ein roter Punkt an der Glocke bedeutet ungelesene Benachrichtigungen. Sie können alle als gelesen markieren.

10 Einstellungen

Profil

Änderung von Name, E-Mail, Telefonnummer.

Sprache

Die App unterstützt 3 Sprachen:

  • 🇩🇪 Deutsch (Standard)
  • 🇵🇱 Polnisch
  • 🇬🇧 Englisch

Erscheinungsbild

  • Hell — weißer Hintergrund
  • Dunkel — dunkler Hintergrund

Sicherheit

Zwei-Faktor-Authentifizierung (2FA) — zusätzlicher Kontoschutz. Nach der Aktivierung müssen Sie einen QR-Code in der Authenticator-App scannen.

Rollenverwaltung (nur Admin)

Der Admin kann:

  • Rollen der Benutzer ändern
  • Individuelle Berechtigungen pro Benutzer festlegen
  • Benutzerdefinierte Rollen erstellen

11 Rollen und Berechtigungen

Jedem Benutzer ist eine Rolle zugewiesen, die bestimmt, was er in der App tun kann.

RolleBeschreibungHauptberechtigungen
Admin Systemadministrator Alles — volle Kontrolle
Geschäftsführung Unternehmensleitung Alles — volle Kontrolle
PM Projektleiter Projekte, Aufgaben, Dateien, Plan, Benutzer. Ohne: GPS, Lagerbearbeitung
BL Bauleiter Aufgaben, Dateien, Kommentare, Lager (Ansicht), Plan (Ansicht). Ohne: Plan-/Lager-/Benutzerbearbeitung
Logistik Logistikabteilung Ansicht von allem + Wochenplan bearbeiten. Urlaubsgenehmigung
Einkauf Einkaufsabteilung Ansicht von allem + Lager bearbeiten
Lagerist Lagermitarbeiter Ansicht von allem + Lager bearbeiten
Büro Büromitarbeiter Ansicht von allem. Ohne Bearbeitung
Arbeiter Bauarbeiter Dashboard, Plan (Ansicht), Kommentare, Aufgabenstatus ändern, Dateien hochladen
Der Admin kann individuelle Berechtigungen für jeden Benutzer festlegen, unabhängig von der Rolle.

12 GPS-Tracking

Das GPS-System ermöglicht die Standortverfolgung von Mitarbeitern im Außendienst.

GPS aktivieren

  1. Gehen Sie zum Benutzerprofil.
  2. Scrollen Sie zum Bereich „GPS-Tracking".
  3. Aktivieren Sie den GPS-Schalter.

Was wird angezeigt

  • Letzte Position — Karte mit Pin + Adresse + Zeitstempel
  • Standortverlauf — Liste der Positionen eines ausgewählten Tages
  • Link zu Google Maps — öffnet die Position in Google Maps
GPS funktioniert nur, wenn die App geöffnet ist. Der Mitarbeiter muss die Standortfreigabe auf dem Handy aktiviert haben.

Wer sieht GPS

Zugriff auf GPS haben: Admin und Geschäftsführung. Andere Rollen sehen den GPS-Bereich nicht.
